# Windows 便携包构建说明

## 构建

```powershell
python -m pip install -r requirements.txt -r requirements-dev.txt -r requirements-build.txt
powershell -File scripts\build-portable.ps1
```

脚本执行测试、PyInstaller one-folder 构建、文档与固定 fixture 复制、ZIP 压缩和 SHA-256 输出。生成目录 `dist` 与中间目录 `build` 已被 Git 忽略。

## 包内入口

```text
启动云端实时转写和AI答案.bat
一键Mock演示.bat
MeetingAICopilot\MeetingAICopilot.exe
```

启动脚本优先使用包内 EXE，不安装 Python。菜单不会后台静默启动真实录音。

## 包内配置与数据

包内只提供空 Key 示例。不要把个人 `config.json` 打入 ZIP。运行数据默认写到当前 Windows 用户桌面，不写进程序目录。

## clean-profile smoke

```powershell
powershell -File scripts\test-portable.ps1 -ZipPath dist\meeting-ai-copilot-1.1.0-rc.1-win-x64.zip
```

通过标准：版本、EXE smoke、固定 fixture Mock、结构化会话、导出和一键启动全部退出 0；应用不依赖系统 Python。

有真实音频设备的 Windows 主机可再加 `-IncludeWindowsAudio`，让包内 EXE 播放固定 fixture 并验收 loopback、麦克风、混合、暂停和热切换。CI 虚拟机通常没有音频设备，因此不使用该开关。
