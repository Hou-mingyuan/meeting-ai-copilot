# 单启动版打包说明

## 包内给用户看到的入口

只保留一个 bat：

```text
启动云端实时转写和AI答案.bat
```

它负责：

```text
安装依赖
启动云端实时 ASR
启动 Coding Plan 流式答案
```

```text
MeetingLiveTranscriber

```
