# Windows 便携版说明

## 启动

解压 ZIP 后双击 `启动云端实时转写和AI答案.bat`。菜单默认提供固定音频 Mock 演示；真实会议模式需要用户自己的 ASR/AI Key，并在打开音频设备前再次确认采集范围和数据去向。

便携版包含 Python 运行时，不要求系统预装 Python，不注册 Windows 服务，不创建计划任务，不后台静默启动。

## 数据位置

默认会话数据保存在当前用户桌面的 `实时监听` 目录，包括结构化 JSON、TXT、Markdown、运行日志和兼容版转写文件。`config.json` 可通过 `output_directory` 改为其它本地目录。

默认保留 30 天。启动时只清理应用自己创建且超过保留期的 `session-*` 文件；不会删除其它文件。

## 卸载与清理

1. 在 TUI 中按 `Q` 停止，确认程序退出。
2. 删除解压后的便携版目录即可卸载。
3. 如需同时删除会议数据，另行删除桌面 `实时监听` 目录；卸载程序不会擅自删除会议记录。
4. 如曾创建 `config.json`，删除便携版目录前确认其中没有仍需保留的本地配置。

## 签名与发布边界

当前构建没有代码签名证书，因此 EXE **未签名**。构建脚本只在本地生成便携 ZIP，不提交、不推送、不发布。
